## Instruções de Execução dos Testes

Os testes deste projeto foram desenvolvidos usando o framework **JUnit 5**. Como se trata de um projeto Java Puro (sem Maven/Gradle), a execução dos testes deve ser feita por meio de uma IDE (Ambiente de Desenvolvimento Integrado) que ofereça suporte nativo ou extensões para JUnit 5 (como IntelliJ, Eclipse ou VSCode).

### Passos para execução na IDE:
1. Abra o diretório raiz deste repositório na sua IDE de escolha.
2. Certifique-se de que a pasta `src` esteja configurada como o diretório de arquivos fonte (*Source Root*).
3. Para executar **todos os testes de todos os casos**, navegue até o arquivo `src/tppe/deduplication/AllCasesSuite.java`.
4. Clique com o botão direito no arquivo e selecione a opção **Run** (ou **Run 'AllCasesSuite'**).
5. A IDE resolverá automaticamente as dependências de JUnit 5 necessárias e exibirá os resultados em sua interface de testes.

### Execução individual por caso:
- Para executar os testes de um caso específico, navegue até o arquivo de teste correspondente (ex: `Case1Test.java`, `Case2Test.java`, etc.) ou sua suíte (ex: `Case1Suite.java`, `Case2Suite.java`, etc.) e execute-o da mesma forma.
